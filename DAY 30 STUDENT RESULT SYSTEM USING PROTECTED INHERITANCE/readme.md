# Day 30 - C++ Protected Access Modifier

## 🚀 Topic

Protected Access Modifier in C++

## 📌 Description

This project demonstrates how the protected access modifier works in C++. It shows how data members can be accessed inside derived (child) classes but not directly from outside.

## 🧠 Concepts Used

* Classes & Objects
* Inheritance
* Protected Access Modifier
* Multi-level Inheritance

## 💻 Implementation

* `Person` class → name, age (protected)
* `Student` class → inherits Person + total marks
* `Result` class → calculates percentage and displays result

## 📷 Output

Displays:

* Name
* Age
* Total Marks
* Percentage

## 🔥 Learning Outcome

Understood how protected members work and how they help in sharing data between parent and child classes safely.
