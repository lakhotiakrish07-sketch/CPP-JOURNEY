# Day 35 - C++ Constructors in Derived Class

## 🚀 Topic

Constructors in Derived Class

## 📌 Description

This project demonstrates how constructors work in inheritance. It shows how a derived class calls the constructor of its base class using an initializer list.

## 🧠 Concepts Used

* Classes & Objects
* Inheritance
* Constructors
* Initializer List

## 💻 Implementation

### 1. Student Info System

* Person → name
* Student → marks

### 2. Employee Salary System

* Employee → id
* Salary → salary

### 3. Vehicle Details System

* Vehicle → brand
* Car → speed

## 📷 Output

Displays data initialized using constructors from both parent and child classes.

## 🔥 Learning Outcome

Understood how constructors are called in inheritance and how to pass values from derived class to base class.
