# Number Menu Program (C++)

This is a simple C++ console program based on basic concepts like loops and conditions.

## What This Program Does

- Prints numbers from 1 to N  
- Prints even numbers till N  
- Generates multiplication table of a number  

## Concepts Used

- Variables  
- User input (`cin`)  
- `if-else` statements  
- `for` loops  
- Arithmetic operators  

## How to Run

Compile the program:

```
g++ main.cpp -o main
```

Run the program:

```
./main
```
