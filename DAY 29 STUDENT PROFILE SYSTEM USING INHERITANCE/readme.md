# Day 29 - C++ Inheritance

## 🚀 Topic

Basics of Inheritance (Public)

## 📌 Description

This project demonstrates how a child class inherits properties from a parent class using public inheritance. It also shows how to call a parent constructor from a child class.

## 🧠 Concepts Used

* Classes & Objects
* Inheritance
* Constructor Calling (Initializer List)

## 💻 Implementation

* `Person` class → id, name, age
* `Student` class → inherits Person + marks
* Input marks and display full details

## 📷 Output

Displays:

* ID
* Name
* Age
* Marks

## 🔥 Learning Outcome

Understood how inheritance works and how to pass values from child to parent constructor in C++.
