# Day 36 - C++ Initializer List in Constructor

## 🚀 Topic

Initializer List in Constructor

## 📌 Description

This project demonstrates how initializer lists are used in constructors to initialize variables at the time of object creation.

## 🧠 Concepts Used

* Classes & Objects
* Constructors
* Initializer List
* Basic Inheritance (practice)

## 💻 Implementation

### 1. Student System

* Initialize name and marks using initializer list

### 2. Rectangle System

* Initialize length and breadth
* Calculate area

### 3. Employee System (basic practice)

* Initialize id and salary

## 📷 Output

Displays values initialized directly through constructors.
