# Day 33 - C++ Ambiguity Resolution

## 🚀 Topic

Ambiguity in Multiple Inheritance

## 📌 Description

This project demonstrates ambiguity in multiple inheritance and how to resolve it using the scope resolution operator (::).

## 🧠 Concepts Used

* Multiple Inheritance
* Function & Variable Ambiguity
* Scope Resolution Operator (::)

## 💻 Implementation

### 1. Teacher vs Student

* Same function `role()` in both classes
* Resolved using `ClassName::function()`

### 2. Dual Salary System

* Same function `salary()` in both classes
* Accessed using scope resolution

### 3. Device Control System

* Same function `start()` in both classes
* Called specific functions using `::`

## 📷 Output

Displays outputs from specific parent classes without ambiguity.

## 🔥 Learning Outcome

Understood how ambiguity occurs in multiple inheritance and how to resolve it using the scope resolution operator.
