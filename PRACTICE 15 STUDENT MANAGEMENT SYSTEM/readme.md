# Day 39 - Student Management using Pointer to Object

## 🚀 Topic

Pointer to Object & Dynamic Memory (new & delete)

## 📌 Description

This project demonstrates how to use pointers to objects in C++ along with dynamic memory allocation using new and delete.

## 🧠 Concepts Used

* Classes & Objects
* Pointers
* Pointer to Object
* Arrow Operator (->)
* Dynamic Memory Allocation

## 💻 Implementation

* Created a `Student` class with name and marks
* Used `new` to create object dynamically
* Took input using pointer
* Displayed output using `->` operator
* Deleted object using `delete`

## 📷 Output

Displays student name, marks, and status (Topper if marks > 90)

## 📈 Learning Outcome

Understood how to use pointers with objects and manage memory dynamically in C++.
