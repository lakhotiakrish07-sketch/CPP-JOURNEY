# Day 38 - C++ Pointer to Object and Arrow Operator

## 🚀 Topic

Pointer to Object & Arrow Operator (->)

## 📌 Description

This project demonstrates how pointers can be used to access objects in C++ and how the arrow operator simplifies accessing members of an object through a pointer.

## 🧠 Concepts Used

* Classes & Objects
* Pointers
* Pointer to Object
* Arrow Operator (->)
* Dynamic Object Creation (new & delete)

## 💻 Implementation

### 1. Student System

* Pointer used to access name and marks

### 2. Rectangle System

* Used pointer to calculate area

### 3. Employee System

* Created dynamic object using new
* Accessed data using ->
* Deleted object using delete

## 📷 Output

Displays object data accessed through pointers.
