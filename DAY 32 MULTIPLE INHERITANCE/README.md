# Day 32 - C++ Multiple Inheritance

## 🚀 Topic

Multiple Inheritance in C++

## 📌 Description

This project demonstrates multiple inheritance where a class inherits properties and functions from more than one parent class. It shows how different functionalities can be combined into a single class.

## 🧠 Concepts Used

* Classes & Objects
* Inheritance
* Multiple Inheritance

## 💻 Implementation

### 1. SmartPhone System

* Camera → takePhoto()
* MusicPlayer → playMusic()
* Smartphone → inherits both

### 2. Hybrid Car System

* PetrolEngine → petrol info
* ElectricEngine → battery info
* HybridCar → inherits both

### 3. Intern System

* Student → marks
* Employee → salary
* Intern → inherits both and checks eligibility

## 📷 Output

Displays:

* Features from multiple parent classes
* Combined functionalities in child class
* Eligibility based on conditions

## 🔥 Learning Outcome

Understood how multiple inheritance works and how to combine functionalities from different classes into one.
