- Address operator (&)

- Access value using pointer

- Swap two numbers using pointers

## Key Learning

A pointer stores the address of another variable.

Example:

int a = 10;
int *ptr = &a;

ptr stores the address of a  
*ptr gives the value stored at that address  

## Programs Implemented

1. Print value and address of a variable
2. Change value using pointer
3. Swap two numbers using pointers
4. Access array elements using pointer

## Important Understanding

- & gives address
- * is used to dereference
- Changing *ptr changes the original variable
