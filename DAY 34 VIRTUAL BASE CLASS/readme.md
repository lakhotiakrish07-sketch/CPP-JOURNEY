# Day 34 - C++ Virtual Base Class

## 🚀 Topic

Virtual Base Class in C++

## 📌 Description

This project demonstrates how virtual base classes solve the diamond problem in multiple inheritance by ensuring only one copy of the base class is shared.

## 🧠 Concepts Used

* Multiple Inheritance
* Diamond Problem
* Virtual Base Class

## 💻 Implementation

### 1. Student Intern System

* Person → base class
* Student & Employee → inherit Person (virtual)
* Intern → inherits both

### 2. Device Feature System

* Device → base class
* Camera & MusicPlayer → inherit Device (virtual)
* SmartDevice → inherits both

## 📷 Output

Displays data from a single shared base class without ambiguity.

## 📈 Progress

Solved 2 problems to practice virtual base class.

## 🔥 Learning Outcome

Understood how virtual inheritance removes duplication and avoids ambiguity in multiple inheritance.
