# Day 27 - C++ Destructor

## 📌 What I Learned
- Destructor basics
- Memory management using delete
- Avoiding memory leaks

## 🚀 Project
Student Record Manager using dynamic memory.

## ⚙️ Features
- Constructor for initialization
- Destructor for cleanup
- Dynamic allocation of marks

## 🧠 Key Concept
Destructor is automatically called when object is destroyed and helps in releasing resources.

## 📌 Output
Constructor: Krish  
Name: Krish, Marks: 90  
Constructor: Rahul  
Name: Rahul, Marks: 85  
Destructor: Rahul  
Destructor: Krish  

## 💡 Conclusion
Destructors are essential for memory management in C++ and prevent memory leaks.
