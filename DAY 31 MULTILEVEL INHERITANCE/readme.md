# Day 31 - C++ Multilevel Inheritance

## 🚀 Topic

Multilevel Inheritance in C++

## 📌 Description

This project demonstrates multilevel inheritance where a class is derived from another derived class. It shows how data and functions can be accessed across multiple levels.

## 🧠 Concepts Used

* Classes & Objects
* Inheritance
* Multilevel Inheritance

## 💻 Implementation

* `Vehicle` class → brand
* `Car` class → inherits Vehicle
* `Speed` class → inherits Car

## 📷 Output

Displays:

* Brand
* Average
* Top Speed

## 🔥 Learning Outcome

Understood how inheritance works in multiple levels and how child classes can access data from parent and grandparent classes.
